# Mushrooms classification CNN

Этот проект демонстрирует классификацию изображений грибов с использованием сверточной нейронной сети (CNN), реализованной на PyTorch.

## Описание проекта

Проект содержит полный конвейер обработки данных и обучения модели для классификации изображений грибов на 9 различных классов. Модель построена на основе архитектуры CNN с использованием PyTorch и достигает точности около 75% на валидационной выборке.

## Структура проекта

- **Загрузка и предобработка данных**: Создание кастомного класса датасета с обработкой поврежденных изображений
- **Аугментация данных**: Применение трансформаций для улучшения обобщающей способности модели
- **Архитектура модели**: CNN с 4 блоками сверток и полносвязным классификатором
- **Обучение**: Использование оптимизатора AdamW с планировщиком обучения ReduceLROnPlateau
- **Визуализация**: Графики потерь и точности в процессе обучения

## Архитектура модели

Модель `MushroomNet` состоит из:
- 4 блоков сверток (32→64→128→256 фильтров)
- Batch Normalization и ReLU активации
- MaxPooling для уменьшения размерности
- Adaptive Average Pooling
- Полносвязный классификатор с Dropout

Всего параметров: 1,176,489

## Используемые технологии

- **PyTorch** - фреймворк для глубокого обучения
- **TorchVision** - трансформации изображений и утилиты
- **PIL** - работа с изображениями
- **Matplotlib** - визуализация результатов
- **NumPy** - численные вычисления

## Результаты

- Финальная точность на валидации после 100 эпох: 75.50%

## Особенности реализации

- Обработка поврежденных изображений с помощью кастомного датасета
- Использование аугментаций (RandomHorizontalFlip, RandomRotation)
- Нормализация по статистике ImageNet
- Эффективная загрузка данных с prefetching и pin_memory
- Регуляризация с помощью Dropout и Weight Decay
